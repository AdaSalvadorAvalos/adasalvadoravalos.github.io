Projects & Research

Master's Thesis

Freesound Loop Generator

Synthesis of new audio loops using neural networks, with a focus on creative sound generation through latent space manipulation. This project explores creative audio loop generation using neural networks, leveraging the RAVE model and the Freesound Loop Dataset. It includes preprocessing pipelines, dataset analysis, training setup, inference scripts in Max for real-time generation, and evaluation tools.

MusicXML Transcriber & Editor

A desktop app that converts piano monophonic audio (MP3/MIDI) into MusicXML, with a web-based interface for editing, playback, and applying musical effects. The system transcribes audio into MusicXML scores and provides tools for visualization, editing, and applying musical effects.

Research Project

Conditioned Morpher Transformer

As part of the Freesound Loop Generator experiments, I developed the Conditioned Morpher Transformer (CMT), a custom transformer model for audio loop morphing with explicit control over style and tempo. The CMT enables smooth interpolation between audio loops while controlling perceptual attributes.

Python PyTorch Machine Learning Audio DSP

Developed a series of hands-on ESP32 projects for a Digital Processors course, including WiFi audio streaming, serial communication, SD card playback, and OLED menu systems.
